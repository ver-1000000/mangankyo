# Mangankyo | 万顔鏡
万顔鏡は、Webカメラの画像をソースにし、
万華鏡のようなうつくしい幾何学パターン画像を表示するWebアプリケーションです。

## 動作環境
以下の環境での動作を確認しています。

- Linux
  - Google Chrome 85.0.4183.102
  - Firefox 80.0.1
- Android
  - Google Chrome 85.0.4183.127
- iPhone
  - iOS 13.7 Safari

また、以下の環境では動作しないことを確認しています。
- iPhone
  - iOS 14.0.1 Safari
  - iOS 14.0.1 Google Chrome
  - iOS 13.7 Google Chrome
